1.设置IP

sudo route add default gw 202.197.75.55

3.设置DNS

sudo nano /etc/resolv.conf

添加nameserver 114.114.114.114

4.重启网络

sudo /ect/init.d/networking restart

或者sudo ifconfig ens3 down

5.测试网络
