Visual Studio 2017 无法登陆提示脚本错误

2018-09-10

浏览器打开发现脚本没问题。重启多试几次，还是不行，就直接搜这个网址，还真有：VS2017登陆不了,TFS无法连接成功的问题
果然，我的 win7 是 ie8，不过都是用 chrome 上网的（巨硬都是用系统自带的东西，也是服了），马上更新
ie 11 更新包
